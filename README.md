# Jataí

Landing page for a fictional educational Open Finance SDK. The experience demonstrates consent,
synthetic accounts, a simulated Pix payment, and callbacks without connecting to real financial
services or using real customer data.

## Stack

- Astro and React
- Tailwind CSS
- GSAP for the scroll-driven terminal sequence
- Motion for title and interface transitions
- Vitest and Testing Library

## Commands

```sh
npm install
npm run dev
npm test
npm run build
npm run preview
```

The Astro project uses `/jatai/` as its base path. The product, SDK responses, and Mel's guidance
are scripted locally for this academic demonstration.
